# MakeChangeProject

## Description
This program promopts the user for their purchase cost and how much they paid and calculates the amount of change due back in 
<ul>
+ twenty dollar bills, 
+ ten dollar bills, 
+ five dollar bills, 
+ one dollar bills,
+ quarters,
+ dimes,
+ nickels,
+ and pennies.
</ul>

## Technologies Used

	- Eclipse
	- Sublime
	- Git/GitHub
	- Unix CommandLine
	- MacOS Terminal
